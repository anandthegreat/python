Merging & Combining
FunctionFormatExamples
mergeSQL-style joins pd.merge(left, right, on='col', how='inner'|'left'|'right'|'outer', suffixes=('_x','_y'))
# inner join (default): pd.merge(orders, customers, on='cust_id')
# left join, different key names: pd.merge(df1, df2, left_on='id', right_on='user_id', how='left')
# anti-join (rows in left NOT in right): pd.merge(df1, df2, on='id', how='left', indicator=True).query("_merge == 'left_only'")
✓ indicator=True → _merge col shows 'left_only','both','right_only'
concatStack dfs vertically or horizontally pd.concat([df1, df2], axis=0|1, ignore_index=True, keys=['a','b'])
# stack rows (union): pd.concat([jan, feb, mar], ignore_index=True)
# stack columns side by side: pd.concat([df1, df2], axis=1)
# keep track of origin: pd.concat([df1, df2], keys=['2024', '2025'])
⚠ axis=0 needs same columns; mismatches → NaN
joinIndex-based join df1.join(df2, on='col', how='left'|'inner')
df1.set_index('id').join(df2.set_index('id'), how='inner')
GroupBy & Aggregation
FunctionFormatExamples
groupby + aggMulti-function aggregation df.groupby('col').agg({'col2': ['sum','mean'], 'col3': 'count'}) df.groupby('col').agg(alias=('col2', 'sum'))
df.groupby('dept')['salary'].agg(['mean', 'max', 'count'])
# named aggregation (clean output): df.groupby('dept').agg( avg_sal=('salary', 'mean'), headcount=('emp_id', 'nunique') ).reset_index()
✓ named agg avoids MultiIndex columns
groupby + transformBroadcast agg back to df df.groupby('col')['col2'].transform('mean'|func)
# add dept avg as new column (same row count): df['dept_avg_sal'] = df.groupby('dept')['salary'].transform('mean')
# pct of group total: df['pct'] = df['sales'] / df.groupby('region')['sales'].transform('sum')
✓ key difference from agg: same shape as input
pivot_table2D cross-tab aggregation df.pivot_table(values, index, columns, aggfunc, fill_value)
df.pivot_table(values='sales', index='region', columns='month', aggfunc='sum', fill_value=0)
rank / cumsumWindow-style calcs df.groupby('col')['col2'].rank(method='dense', ascending=False) df['col'].cumsum()
# rank within group (dense = no gaps): df['rank'] = df.groupby('dept')['salary'].rank(method='dense', ascending=False)
df['running_total'] = df.groupby('user')['spend'].cumsum()
